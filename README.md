# Bus Reservation System

## Project Overview

The Bus Reservation System is a web-based application developed to simplify bus ticket booking and management. The system allows administrators to manage buses, routes, and schedules while enabling users to view available buses and make reservations efficiently.

## Features

### Admin Module

* Add New Buses
* Update Bus Details
* Delete Bus Information
* Manage Routes
* View Customer Feedback
* Monitor Bookings

### Route Management

* Add Routes
* Edit Routes
* Delete Routes
* View Route Information

### Booking Management

* Search Available Buses
* Book Tickets
* View Booking Details
* Manage Passenger Information

### Feedback Module

* Submit Feedback
* View Customer Reviews

## Architecture

The project follows the MVC (Model-View-Controller) architecture:

* Model: Entity classes and database mapping
* View: JSP pages for user interface
* Controller: Spring MVC Controllers handling requests
* Service Layer: Business logic implementation
* Repository Layer: Database operations

## How to Run

1. Clone the repository.
2. Import the project into Eclipse or Spring Tool Suite.
3. Configure MySQL database.
4. Update database credentials in application.properties.
5. Run the Spring Boot application.
6. Open the application in a browser.
